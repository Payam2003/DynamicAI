# DynamicAI

Progetto basato sulla creazione di un'interfaccia LLM non-chat-based.

Per avviare il server Vite, andare nel folder /DynamicAI/DynamicAI e da terminale eseguire il comando:
```
npm run dev
```

Se non si dispone delle librerie necessarie, eseguire: ``` npm install ```

Invece per eseguire l'envoiroment del server, dal folder /DynamicAI/chatbot eseguire:

```
uvicorn app.main:app --reload
```
